# aethergraph-cli-dist

Public distribution of the **AetherGraph CLI** compiled binaries.

This repository holds **only the compiled release binaries** for the AetherGraph
CLI. The source code lives in a separate repository under the
[AetherGraph Source-Available License](https://github.com/alduron/aethergraph-cli/blob/main/LICENSE)
(source-available, not open source - published so you can audit what runs on your
machine).

## Install

Use the one-line install command shown on your project's **Settings** page in the
AetherGraph dashboard. It downloads the binary from the latest release here and
runs `aethergraph onboard` to wire your project.

Releases here are mirrored from the CLI build pipeline; each asset ships with a
`.sha256` checksum.
